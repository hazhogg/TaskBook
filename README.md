# ✅ To-Do List

A simple and clean command-line to-do list manager written in Python. Add, view, edit, and remove tasks through an easy-to-use menu — perfect for keeping track of what you need to get done today.

---

## Features

- **View tasks** — see your full task list numbered and neatly formatted
- **Add tasks** — quickly add a new task with a description
- **Edit tasks** — update any task's description, with the current value shown as a default
- **Remove tasks** — delete a task by selecting its number
- **Personalised greeting** — enter your name at the start for a personal touch

---

## Getting Started

### Prerequisites

- Python 3.x (no external libraries required)

### Installation

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### Running the App

```bash
python todo.py
```

---

## Usage

Enter your name when prompted, then choose from the menu:

| Option | Action |
|--------|--------|
| 1 | Show all current tasks |
| 2 | Add a new task |
| 3 | Edit an existing task |
| 4 | Remove a task |
| 5 | Exit |
